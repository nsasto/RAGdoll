# RAGdoll v1

**Location:** `ragdoll_v1/`

**Purpose:**  
Legacy or alternative implementation of RAGdoll with its own set of modules.

**Key Components:**

- `chatterbox.py`, `config.py`, `helpers.py`, `index.py`, `models.py`, `multi_query.py`, `prompts.py`, `retriever.py`, `scraper.py`

**Features:**

- Self-contained RAG pipeline.
- Custom retriever and prompt management.
- Web scraping and multi-query support.
