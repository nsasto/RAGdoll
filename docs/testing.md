# Testing

**Location:** `tests/`

**Purpose:**  
Automated tests for all modules.

**Key Components:**

- `test_*.py`: Unit and integration tests for each feature.
- `conftest.py`: Pytest configuration.
- `test_data/`: Sample files for testing (PDF, DOCX, PPTX, XLSX, TXT).

**Features:**

- Comprehensive test coverage.
- Test data for document processing.
- Pytest-based test suite.
