# Examples

**Location:** `examples/`

**Purpose:**  
Jupyter notebooks and scripts demonstrating usage of core features.

**Key Components:**

- `chunker.ipynb`, `embeddings.ipynb`, `ingestion.ipynb`: Step-by-step guides.
- `config_example.py`, `tools.py`: Example scripts.
- `graph_retriever_example.py`: Demonstrates `Ragdoll.ingest_with_graph()` and how to query the resulting graph retriever.

**Features:**

- Hands-on tutorials for each feature.
- Ready-to-run code for experimentation, including the new hybrid graph/vector retrieval path.
